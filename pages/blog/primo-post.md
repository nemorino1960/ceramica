# Il Mio Primo Post

Benvenuto nel blog! Questa pagina si trova in una sottocartella.

## Struttura Annidata

Questo file Markdown si trova in `pages/blog/primo-post.md` e verrà generato come `blog/primo-post.html` nella cartella di output.

## Vantaggi

- **Organizzazione migliore**: Puoi organizzare i contenuti per categoria
- **URL puliti**: Gli URL riflettono la struttura delle cartelle
- **Navigazione intuitiva**: Facile capire dove si trova ogni pagina

## Link Relativi

I link agli assets (CSS, JS) e alle altre pagine vengono gestiti automaticamente dal generatore, quindi funzionano correttamente indipendentemente dalla profondità della cartella.
