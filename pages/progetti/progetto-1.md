# Progetto 1: Ceramica

Questo è un esempio di pagina di progetto in una sottocartella dedicata.

## Descrizione

Ceramica è un generatore di siti statici scritto in Python che supporta:
- Markdown
- Template Jinja2
- Strutture annidate
- Build multiple (debug/deploy)

## Tecnologie

- **Python 3.12+**
- **Markdown** per i contenuti
- **Jinja2** per i template
- **CSS3** per lo styling
