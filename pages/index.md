## Caratteristiche

- **Semplice**: Scrivi in Markdown, ottieni HTML
- **Configurabile**: Personalizza tutto tramite file JSON
- **Flessibile**: Template HTML personalizzabili
- **Deploy facile**: Cartelle separate per debug e produzione

## Come funziona

1. Scrivi le tue pagine in Markdown nella cartella `pages/`
2. Personalizza i template nella cartella `template/`
3. Configura il sito in `config/config.json`
4. Esegui `python build.py` per generare il sito
5. Trova i file pronti in `deploy/` o `debug/`
