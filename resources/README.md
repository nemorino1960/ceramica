# Risorse Aggiuntive

Questa cartella contiene risorse che verranno copiate direttamente nelle cartelle di output (debug/deploy).

Puoi inserire qui:
- File robots.txt
- Favicon
- Sitemap.xml
- File .htaccess
- Altri file statici che non necessitano di processamento

Tutti i file e le cartelle qui contenuti verranno copiati nella root del sito generato.
